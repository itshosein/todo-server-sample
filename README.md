### Installation

<div id="installation"></div>

1. Clone the repo

   ```sh
   git clone git@github.com:itshosein/todo-server-sample.git
   ```

2. Install NPM packages
   ```sh
   npm install
   ```
3. build the project

   ```sh
   npm run build
   ```
4. run the main script

   ```sh
   npm start
   ```
